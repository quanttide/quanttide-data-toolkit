# 文档写作风格

## 标题

- 文档首行 `# 领域模型`，不带包名
- 模型标题 `## 中文名\`TypeName\``，如 `## 管道\`Pipeline\``
- 枚举用 `###`，如 `### 任务状态\`TaskStatus\``

## 字段条目

按分类分组，分类头单独一行：

```
标识
- 中文名`fieldName`（`Type`）：描述。
- 中文名`fieldName`（`Type`）：描述，默认 `value`。
```

规则：
- 中文名与 backtick 间无空格
- 类型用全角括号 `（`Type`）`
- 描述用全角冒号 `：`
- 句尾句号
- 默认值表述：`，默认 `value`。`

## 枚举

- 首句概括：`枚举值。` 或 `枚举值。属性包括：`
- 属性条目同上格式，设计意图用 `——` 引出
- 表前过渡句：`标准属性如下：`
- 表头用 backtick 括字段名，如 `| `name` | `label` | `color` |`
